# Add Lazy Load Attributes to All Img Tags in WordPress

This repository provides a simple method to add lazy load attributes to all img tags in WordPress.

## Usage

To add lazy load attributes to all img tags in WordPress, follow these steps:

1. copy function lazy load.

2. Open the `functions.php` file in your theme for editing.

3. paste code in 'function.php'.

4. Save the changes.

5. That's it! Now, all img tags in your WordPress site will have lazy load attributes.
